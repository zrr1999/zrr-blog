# BoneTeX

BoneTeX Code 是一个高效的LaTeX文档编写工具，可以大幅度提高编写文档的效率和质量，其分为三部分，BoneTeX Core、BoneTeX Server和BoneTeX Editor。BoneTeX Core是一个自动生成大量繁琐重复的高质量LaTeX代码的Python库；BoneTeX Server是一个基于Flask的后端接口脚本，使BoneTeX Core的核心功能与本地环境无关；BoneTeX Editor是一个基于Electron的BoneTeX语言专用编辑器。
